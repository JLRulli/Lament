## Overview

Lament's world design follows a linear progression through distinct atmospheric zones, each contributing to the game's emotional journey and dark fantasy aesthetic with Lynchian surrealist elements. Players traverse self-contained worlds, each composed of interconnected levels that build toward a climactic boss encounter.

**Design Philosophy**:
- Each world has unique visual identity and emotional tone
- Surrealist elements blend fantasy with dreamlike, unsettling imagery
- Worlds support both precision platforming and bullet-hell combat
- Environmental storytelling conveys loss, tragedy, and corrupted beauty
- Linear progression (not metroidvania-style backtracking)

## Surrealist Design Principles

**Lynchian Elements to Consider**:
- Familiar objects in unfamiliar contexts
- Scale distortions (giant everyday objects, tiny architecture)
- Dreamlike logic (water flowing upward, rooms bigger on inside)
- Temporal distortions (past/present bleeding together)
- Sound design: Wrong sounds for actions, uncanny audio
- Unsettling beauty (gorgeous visuals with disturbing undertones)

**Reference**:
- **David Lynch films** - Eraserhead, Twin Peaks (surreal imagery, dreamlike logic)
- **Yume Nikki** - Surreal exploration, dreamworld logic
- **Inside/Limbo** - Environmental storytelling through unsettling imagery

**Design Questions**:
- [ ] How surreal is too surreal? (Maintain gameplay clarity)
- [ ] Which surrealist elements support themes vs distract?
- [ ] How to telegraph surreal mechanics vs pure aesthetic?
- [ ] Does surrealism escalate with world progression?

## Design Philosophy Summary

**World Design Supports Game Pillars**:

1. **Emotional Consequences**: Worlds reflect tragedy, loss, corruption - mirrors companion sacrifice theme
2. **Mastery Through Empathy**: Beautiful but dangerous worlds motivate careful, skilled play
3. **Surreal Fantasy**: Lynchian elements make familiar fantasy unsettling and dreamlike
4. **Tight Combat Flow**: World design supports both platforming precision and bullet-hell combat

**Success Criteria**:
- Each world feels distinct and memorable
- Environments support both platforming and combat
- Surrealist elements enhance rather than confuse
- Visual storytelling conveys tragedy without words
- World aesthetics align with emotional journey
- Linear progression maintains forward momentum
